# Arduino 7Segment

Custom Arduino Library for creating 7-Segment displays using addressable RGB stripes.

## Requirements

 - Adafruit Neopixel

## Usage

 1. Clone to Arduino Library Folder

## Development

### Requirements
 - cmake (3.27 or higher)

### Run Tests
 
 1. Clone Repo
 2. Run `chmod +x buildAndTest.sh`
 3. Run `./buildAndTest.sh`
